# Página Inicial de Navegador Personalizada

Uma página inicial de navegador simples e customizável, projetada para substituir a página padrão de "Nova Aba". Este projeto oferece uma interface limpa com funcionalidades úteis para o dia a dia.

## ✨ Funcionalidades

*   **Relógio e Data Dinâmicos:** Exibe a hora e a data atuais.
*   **Barra de Pesquisa Google:** Permite realizar pesquisas diretamente na página.
*   **Atalhos Rápidos:** Acesso fácil a sites como Gemini, Gmail, YouTube e GitHub.
*   **Alternador de Tema:** Suporte para tema claro (light) e escuro (dark), com persistência da escolha do usuário.
*   **Ícones Modernos:** Utilização do Font Awesome para ícones.

## 🛠️ Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript
*   Font Awesome

## 🚀 Como Usar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` no seu navegador.
3.  Para definir como sua página de nova aba:
    *   **Chrome:** Utilize extensões como "New Tab Redirect" e configure a URL para o caminho local do `index.html` (ex: `file:///C:/caminho/para/seu/projeto/.browser-start-page/index.html`).
    *   **Firefox:** Acesse `about:config`, procure por `browser.newtab.url` e defina o valor para o caminho local do `index.html` (ex: `file:///C:/caminho/browser-start-page/index.html`).
