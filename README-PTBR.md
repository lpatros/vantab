## Descrição

Uma página inicial de navegador simples e customizável, projetada para substituir a página padrão de "Nova Guia". Este projeto oferece uma interface limpa com funcionalidades úteis para o dia a dia.

## Como Usar

### Instalação da Extensão

1.  Clone este repositório ou faça o download do código.

2.  **Chromium**
    *   Abra o navegador e acesse `chrome://extensions/` (Chrome).
    *   Ative o **Modo de desenvolvedor** (toggle no canto superior direito).
    *   Clique em **Carregar sem compactação** (ou "Load unpacked").
    *   Selecione a pasta do projeto onde está o arquivo `manifest.json`.
    *   A extensão será instalada e a página personalizada aparecerá automaticamente ao abrir uma nova guia.
