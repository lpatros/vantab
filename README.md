## Description

A simple and customizable browser homepage designed to replace the default "New Tab" page. This project offers a clean interface with useful features for your daily routine.

## How to Use

### Extension Installation

1.  Clone this repository or download the code.

2.  **Chromium**
    *   Open your browser and go to `chrome://extensions/` (Chrome).
    *   Enable **Developer mode** (toggle in the top right corner).
    *   Click on **Load unpacked**.
    *   Select the project folder containing the `manifest.json` file.
    *   The extension will be installed, and the custom page will appear automatically when you open a new tab.
